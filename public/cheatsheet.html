TodUx API

API Documentation

Base URL: /api

REST API for Todo Management

Complete API for managing todo lists and user accounts with JWT authentication, MongoDB storage, and secure password handling.

Technology Stack

Framework: Node.js, Express.js

Database: MongoDB with Mongoose

Authentication: JWT

Security: bcryptjs, ArcJet

Endpoints

  • POST /auth/sign-up - Register new user
  • POST /auth/sign-in - Login user
  • GET /todos - List todos
  • POST /todos - Create todo
  • PUT /todos/:id - Update todo
  • DELETE /todos/:id - Delete todo
  • GET /users - Get user profile
  • PUT /users - Update user
  • DELETE /users - Delete account

Authentication

API uses JWT for authentication. Include token in Authorization header for protected endpoints.

Authorization: Bearer <token>

Response Format

Standard Response { "success": true, "message": "description", "data": {} }
